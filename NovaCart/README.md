# NovaCart 🚀

A modern, professional ecommerce front-end built with **HTML5, CSS3, Vanilla JS, and Bootstrap 5**.
27 pages · Shared components · Fully responsive · Dark, premium UI.

## Structure
```
NovaCart/
├── index.html · login · signup · forgot-password · verify-otp · reset-password
├── home · products · product-details · categories · search
├── cart · checkout · payment · payment-success · orders · order-details
├── wishlist · compare
├── profile · edit-profile · address · notifications
├── support · about · contact · faq
├── css/style.css      (design system + all components)
├── js/app.js          (navbar/footer injection, mock data, interactions)
├── images/  assets/  api/
```

## Run
Just open `index.html` in a browser. No build step required.
Or serve locally: `python3 -m http.server 8000`
